# 🌊 Shahriar Jaman — Ocean Blue Portfolio

A personal portfolio website built with pure HTML & CSS, featuring a deep ocean blue theme with smooth animations and a fully responsive layout.

---

## 🎨 Design — Ocean Blue Theme

- **Deep navy background** with soft blue glowing orbs for an immersive underwater feel
- **Floating transparent bubbles** that rise up the page as an animated background effect
- **Gentle animated wave strip** at the bottom of the viewport
- **High-contrast white & light blue text** — carefully chosen for maximum readability
- **Elegant typography** — *Playfair Display* for headings + *DM Sans* for body text

## ✨ Features & Extras

- 🔝 **Fixed navigation bar** at the top with smooth scroll links
- 🎞️ **Scroll-triggered animations** — sections fade in as you read down
- 💡 **Card hover effects** with blue glow and lift animation
- 📱 **Fully responsive** — works great on mobile, tablet, and desktop
- 🫧 **Bubble particle system** — 22 animated bubbles float upward
- 🌊 **SVG wave animation** rendered at the bottom of the screen
- 🖱️ **Skill progress bars** with animated fill on load

---

## 🛠️ Built With

- **HTML5** — structure and content
- **CSS3** — styling, animations, and layout (Grid & Flexbox)
- **Vanilla JavaScript** — bubble generation & scroll reveal
- **Google Fonts** — Playfair Display, DM Sans, Space Mono

---

## 🚀 How to Use

1. Download the `shahriar-ocean-portfolio.html` file
2. Open it in any modern web browser (Chrome, Firefox, Edge, Safari)
3. No installation or internet connection required — it works offline!
